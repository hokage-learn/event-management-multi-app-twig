=============================================
GETTIX - STATIC HTML APPLICATION
=============================================

WHAT TO DEPLOY:
---------------
Deploy ALL files and folders in this directory to your web server.

QUICK START:
------------
1. Upload all files to your web server
2. Point your domain to this directory
3. Access via browser

FILES INCLUDED:
---------------
- index.html        (Main entry point)
- css/             (Stylesheets)
- js/              (JavaScript modules)
- assets/          (Images and SVGs)
- _redirects       (Netlify config)
- .htaccess        (Apache config)

TECHNICAL NOTES:
----------------
- Uses hash-based routing (#/dashboard, #/tickets)
- No build process required
- Works on any static web host
- Requires modern browser (ES6+)

DEPLOYMENT OPTIONS:
-------------------
- GitHub Pages
- Netlify
- Vercel
- Surge.sh
- Traditional web hosting (cPanel, FTP, etc.)

For detailed deployment instructions, see:
../DEPLOYMENT.md
